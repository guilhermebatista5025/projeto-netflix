# 🎬 Netflix Clone

Uma réplica moderna da interface da Netflix desenvolvida com foco em frontend, trazendo uma experiência visual imersiva, responsiva e próxima de uma aplicação real de streaming.

---

## 📺 Sobre o projeto

O **Netflix Clone** é uma aplicação frontend que simula a interface da Netflix, exibindo filmes e séries em um layout dinâmico e organizado por categorias.

O projeto foi desenvolvido com o objetivo de praticar construção de interfaces profissionais, consumo de API e experiência do usuário.

---

## ✨ Funcionalidades

* 🎥 Listagem de filmes e séries
* 📂 Categorias (Populares, Tendências, Top Rated)
* 🔎 Barra de busca
* 🎬 Banner principal dinâmico
* 📱 Layout responsivo (mobile e desktop)
* 🖱️ Efeitos de hover e animações
* 📄 Modal com detalhes do filme

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla ou React, se aplicável)
* API (TMDB)

---

## 🎨 Destaques

* Interface inspirada na Netflix original
* UI moderna com foco em usabilidade
* Estrutura organizada e escalável
* Uso de dados dinâmicos via API
* Experiência fluida e interativa
